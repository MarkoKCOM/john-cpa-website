**Nobody is building an AI agent that autonomously chases clients for documents via WhatsApp on behalf of the CPA.**

| Capability | Liscio | ClientWindow | FinBot | Autoflow | **HeshbonAI** |

| WhatsApp-native | No | Yes (bridge) | Support only | No | **Yes** |
| Autonomous doc chasing | No | No | No | Partial | **Yes** |

| Transparent pricing | No | Yes | No | No | **Yes** |

### 5 Gaps to Exploit

1. **WhatsApp-native AI agent for doc collection** — ClientWindow is dumb pipe, nobody has AI + WhatsApp + accounting context
2. **No portal/app required** — every competitor forces clients onto new platforms. HeshbonAI meets clients on WhatsApp
3. **Israeli market underserved by AI** — FinBot/WeBenefit have basic OCR, no AI agents
4. **Transparent pricing** — most competitors hide behind demos. Show pricing = more qualified leads
5. **Autoflow proves the demand** — they duct-tape Make.com + Google Sheets for this exact workflow. HeshbonAI is the product version
